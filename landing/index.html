核心能力

从采集到输出，一条链路完成；同时保留可追溯的“原始证据”。

长录音分段

每 5 分钟自动保存一段，支持任意时长录音，降低意外丢失风险。

语音转文字

支持 Whisper 及兼容接口，转录完成后可手动修正文案。

AI 总结

一键生成结构化总结，用于快读与复盘；原文作为可追溯来源。

本地存储

会议记录与配置持久化在本地，列表回顾快速、离线可查看。

复制与分享

总结与原文可一键复制或分享，方便同步到协作工具与文档。

可编辑原文

修正识别错误后重新生成总结，让最终输出更可靠、更贴近真实会议。

如何使用

三步完成：配置 → 录音 → 回顾与输出。

1

配置 API

在设置页填入 STT 与 LLM 的 Key；可选配置 TTS 朗读总结。

2

开始录音

点击麦克风开始采集；长会议自动分段保存，安心记录。

3

生成总结并回顾

转录 + 总结完成后，在详情页查看；原文可编辑并重新生成总结。

你真正需要的是“可追溯的总结”
总结用于快读，原文用于核对；两者缺一不可。
